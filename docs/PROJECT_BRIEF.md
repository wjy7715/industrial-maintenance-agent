# 项目说明

## 固定题目

多工具调用工业运维Agent：设备故障查询+维修方案自动生成

## 目标用户与问题

面向需要快速查找设备状态、故障记录和维修资料的运维人员。Agent 将分散信息汇总成可核对的初步排查草稿，不能代替现场诊断和维修授权。

## 第一版范围

- 演示对象：工业离心泵。
- 输入：设备编号和自然语言故障现象。
- 工具：遥测、故障历史、风险规则、维修知识。
- 输出：候选原因、检查步骤、处理建议、安全警告、证据和工具轨迹。
- 权限：只读与草稿；不控制真实设备。

## 完成标准

- CLI 与网页均可运行。
- 工具具有单一职责和结构化输入输出。
- 无匹配时不生成猜测性动作。
- 每条维修知识可追溯。
- 30 个评测案例和自动化测试全部通过。
- Hermes 可选接入，不影响离线降级。
