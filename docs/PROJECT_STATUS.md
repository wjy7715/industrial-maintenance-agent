## 已知限制

- 仿真设备不对应真实工厂；用户上传快照也未经系统独立核验。
- 当前检索是可解释的加权关键词基线，不是向量检索。
- Hermes 未强制安装，避免让无密钥环境失去可运行性。
- 不包含真实设备控制。
- 没有企业身份源，当前权限边界是单机教学环境，不代表生产 RBAC。
- 未完成真实用户访谈、现场数据授权与影子试点，不能宣称生产可用。
