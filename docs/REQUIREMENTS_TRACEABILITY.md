## 本地发布剩余顺序

1. P0 安全缺口：单位契约、权限注册表、输出验证器。
2. P1 数据能力：多时间点遥测趋势和质量摘要。
3. 评测：独立盲测框架、失败案例矩阵、性能基线。
4. 治理：专家审核、知识校验、备份恢复和运行手册。
5. 发布：统一验收、项目总结 PPT、最终 GitHub 推送。
